= heos-dial

This project holds a small controller software for HEOS based on MD5Dial and Rust

== Modules

[cols="1,3"]
|===
| heos-dial
| The actual firmware

| heos-lib
| The HEOS library

| heos-tui
| A TUI based on the library
|===
